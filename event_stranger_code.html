Round 1 – Demogorgon Code Quest 1. The participants will be assigned a computer and the screen of the computers will be turned off. 2. The participants will then be given the problem statement and they will be asked to write the proper code for the execution of the problem statement. 3. Time duration of 30 minutes be allotted to finish the given task. 4. The participants will get the lifelines to review the code and make changes to it at the cost of penalties. 5. The participants who are able to write the right code the fastest and accurate will be qualified for next round.In case there is tie the Qualification will be based on less no. of Errors in Code.

Round 2- GPT Hunt 1. The participants will be provided with a clue/riddle to find problem statement. 2. Solving clues/riddle will lead the participants to a problem statement. 3. The participants will be assigned with a problem statement where they have to code by prompting GPTs. 4. Requirements will be specified to the participants and they will have to create GUI/CLI according to it. 5. Time duration of 60 minutes will be allotted to finish the task. 6. GUI will be more preferred than CLI. 7. The final code should to be submitted to the organising team before the time ends. 8. The team which solves the problem earliest will be the winner and following 2 teams runner ups respectively
